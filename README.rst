Install
-------------------------------------------------------------------------------

``diablo2_doc`` is released on PyPI, so all you need is:

.. code-block:: console

	$ pip install diablo2_doc

To upgrade to latest version:

.. code-block:: console

	$ pip install --upgrade diablo2_doc
