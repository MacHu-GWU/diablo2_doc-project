战网角色Build
===============================================================================

在暗黑2中, Build就是打造一个具有特定功能的角色所需要的 装备, 属性点, 技能点, 雇佣兵. 例如 MF法师, KC亚马逊, PK死灵法师, Bus圣骑士.
