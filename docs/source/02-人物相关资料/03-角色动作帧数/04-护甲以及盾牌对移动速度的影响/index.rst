护甲以及盾牌对移动速度的影响
==============================================================================

.. contents::
    :local:

盔甲:

- 轻型盔甲: 跑速不变
- 中型盔甲: 跑速减慢5%
- 重型盔甲: 跑速减慢10% 精力消耗速度加倍

盾牌:

- 轻型盾牌: 跑速不变
- 中型盾牌: 跑速减慢5%
- 重型盾牌: 跑速减慢10%
